## 文档规范

- **数据**：原始导出数据（Excel、CSV 等），按平台+日期归档
- **汇总报告**：定期（月/季度）汇总分析报告，含图表与结论
- **其他**：流程文档、会议纪要、竞品分析等辅助材料

## 命名规范

文件命名格式：YYYY-MM-DD_类型_内容简述

示例：2026-03-31_月报_塞控电气_阿里国际站.xlsx

## 使用说明

1. 按公司 → 平台 → 文档类型三级目录归档
2. 原始数据保持只读，报告类文件可自由编辑
3. 定期（月/季度）更新汇总报告，保持数据时效性
