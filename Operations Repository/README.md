## 文档规范

- **数据**：原始导出数据（Excel、CSV 等），按平台+日期归档
- **汇总报告**：定期（月/季度）汇总分析报告，含图表与结论
- **其他**：流程文档、竞品分析、会议纪要等辅助材料

## 命名规范

YYYY-MM-DD_平台_内容简述.ext

示例：2026-03-15_阿里国际站_询盘.xlsx
